# Borges Incorporadora — Site (GitHub Pages)

## Conteúdo
- `index.html` (One Page)
- `assets/logo-borges.png` (logo tratada: fundo branco e recorte limpo)
- `assets/img/` (imagens geradas)
- `CNAME` (domínio personalizado)

## Publicar no GitHub Pages
1. Crie um repositório no GitHub.
2. Envie estes arquivos para a branch `main`.
3. GitHub: **Settings → Pages** → Deploy from a branch → `main` /(root)

## Domínio
O arquivo `CNAME` já aponta para `borgesincorporadora.com.br`.
Depois de configurar o DNS e propagar, ative **Enforce HTTPS** no GitHub Pages.

## Formulário de contato
Implementado via **Google Forms** em um modal (embed). Funciona 100% em GitHub Pages e salva respostas no Google Sheets.
